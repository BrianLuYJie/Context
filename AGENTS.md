# 预订管理重构实现

## 内容

- `index.html` — 预订管理页面右侧内容区域的 1:1 静态 HTML/CSS/JS 还原，包含基础交互。
- `assets/figma-full-page.png` — 从 Figma 节点导出的整页参考图，用于保留左侧导航和顶部导航截图。
- `assets/user-reference.png` — 用户提供的参考截图备份。

## 使用

直接用浏览器打开 `index.html` 即可预览，不需要安装依赖或启动服务。

## 交互范围

- 课程类型 Tab 切换
- 门店选择
- 查询按钮反馈
- 上一周/下一周反馈
- 有数据的课程块 hover 高亮为 `#E8F3FF`
- 课程块无点击弹窗、无点击描边
